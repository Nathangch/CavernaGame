# Heaven or Hell - Julgamento Final ⚖️

Um jogo de julgamento moral estilizado feito com **Python** e **Pygame**. Decida o destino de almas com base em suas boas ações e crimes, levando em conta a gravidade de cada ato!

## 🎮 Como Jogar
1.  Certifique-se de ter o Python instalado.
2.  Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
3.  Execute o jogo:
    ```bash
    python main.py
    ```

## 🌟 Funcionalidades
- **Sistema de Pesos Morais**: Nem todo crime é igual! Assassinatos pesam muito mais que pequenas mentiras.
- **Ambientação Dinâmica**: Efeitos de fogo procedurais no Inferno e nuvens divinas no Céu.
- **Diversidade de Almas**: Bonequinhos estilizados com variações femininas e masculinas.
- **Estética Modernizada**: Menu principal, sistema de pontuação e combo, e layout responsivo.

## 🛠️ Tecnologias
- **Python 3.x**
- **Pygame** (Para lógica de jogo e renderização)
- **Cálculos Procedurais** (Para animações e personagens)
